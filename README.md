| Coin Type | Symbol | Coin    |
|----------:|--------|---------|
|         0 | BTC    | Bitcoin |

## Usage

! Do not use IDE to test, command ascii will not work properly.

```shell
mvn clean
mvn package
java -jar target/dtc-crypto-cli-0.1.0-SNAPSHOT-jar-with-dependencies.jar
```
